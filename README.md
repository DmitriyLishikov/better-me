## Software requirements

-   PHP >= 8.0.2

-   MySQL 5.7.\*

-   NPM >= v16.8.0 

### 1. Clone project from git

```bash
git clone git@github.com:DmitriyLishikov/better-me.git
```

### 2. Install composer dependencies

```bash
 composer install
```

### 3. Install JS dependencies

```bash
npm install

npm run build
```

### 5. Migrate

```bash
php artisan migrate
```
